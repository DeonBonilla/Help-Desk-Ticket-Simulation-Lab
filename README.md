# 🎫 Help Desk Ticket Simulation Lab

## 📋 Overview

This project documents simulated Help Desk support tickets designed to mirror real-world technical issues commonly encountered in enterprise IT environments. Each ticket follows a structured troubleshooting process, including issue identification, investigation, root cause analysis, resolution, and documentation.

The goal of this lab is to demonstrate practical troubleshooting skills, customer support methodology, and technical documentation practices used by IT Support and Help Desk professionals.

---

## 🛠️ Technologies Used

- Windows 11
- Windows Server 2022
- Active Directory
- Group Policy
- Command Prompt
- Windows Settings
- Event Viewer
- Oracle VirtualBox
